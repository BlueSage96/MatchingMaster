# Matching Master

DESCRIPTION:
Matching Master is a card matching game made in React and Vite. Match all card pairs to win the game.

DEPENDICIES:
react + vite
react-router
prettier
babel
confetti

SETUP:
Use "Settings" to toggle music and sounds on/off
use "npm run dev" to run the game locally
